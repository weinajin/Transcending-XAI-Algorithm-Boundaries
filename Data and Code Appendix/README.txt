The Data and Code Appendix provides the qualitative and quantitative data, and the source python code Jupyter notebook for the quantitative data analysis. Each folder contains the following content:

1. Folder: quantitative_dataset
It has the quantitative data collected from the user study, including participants' demographic information (demography.csv), and their explanation form card ranking data (ExplanationFormRanking.csv).

To use the data, we also provide a document showing the acronym of the corresponding explanation form cards (explanation_form_labels.pdf), and a data description file (End-User-XAI_data_description.txt) to use the data.

2. Folder: quantitative_data_analysis_code
The three Jupyter notebooks contain python code for the quantitative data analysis, including the report on participants' demography (demographic.ipynb), the main analysis code (quantitative_analysis.ipynb), and the code to generate the explanation-goal-specific frequency tables in the Appendix Section 4.1 (graph_on_task_specific_card_ranking.ipynb).

3. Folder: qualitative_analysis_raw_data_coded_text
Each document name is a label (the "code" in qualitative data analysis) on the text transcripts of the interview recordings. The content of each document is the raw qualitative data text with the label. The labels of explanation forms starts with "@", and their full names corresponding to the acronyms are in the quantitative_dataset/explanation_form_labels.pdf.
